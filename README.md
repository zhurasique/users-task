## Installation instruction

1. download project
``
$ git clone https://github.com/zhurasique/users-task.git
``
2. download laravel
3. add database to .env file
4. make migrations
``
$ php artisan migrate
``
5. start project
``
$ php artisan serve
``
6. go to localhost url (http://127.0.0.1:8000/ by default)
